README

This script runs the evolutionary model of hemispheric specialization and saves all results to the Desktop.

Requirements

Python 3.10 or later

Packages:

numpy
pandas
matplotlib
scipy

The output folder includes:

tables of final lineage results
statistical test results
generation traces
main figures
supplementary figures
Main analyses

The script runs:

Stage 1: symmetric development
Stage 2: positive developmental asymmetry
Stage 2R: negative developmental asymmetry
Stage 3_PosSeed: reseeded persistence after positive bias
Stage 3_NegSeed: reseeded persistence after negative bias

Additional analyses include:

symmetry controls
bias sweep
fitness valley analysis
environmental necessity analysis
robustness checks
